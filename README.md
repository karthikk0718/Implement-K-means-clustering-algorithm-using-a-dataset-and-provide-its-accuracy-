📌 K-Means Iris Clustering (GUI Mini Project)

This project implements K-Means clustering on the Iris dataset with a user-friendly GUI using Tkinter. It allows users to load data, find optimal clusters (Elbow Method), perform clustering, and visualize results.

⚙️ Requirements
Python 3.10 or 3.11 (Recommended)

Libraries:
python -m pip install numpy matplotlib scikit-learn scipy

▶️ How to Run
python your_file_name.py

🚀 Features
GUI using Tkinter
Elbow Method for optimal K
K-Means clustering
Accuracy & Confusion Matrix
PCA-based cluster visualization

📌 Output

Displays clustering results, accuracy, confusion matrix, and graphical visualizations.
